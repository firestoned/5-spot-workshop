# Workshop slides

`5-spot-workshop.pptx` — 14 slides with speaker notes, built for the OSFF session.

Before presenting: replace `github.com/<your-org>/5-spot-workshop` on the closing
slide with the real repo URL. The deck is intentionally thin — the lab repo is the
real content; slides are the map and the energy.

Rough flow: problem → what 5-Spot is → lifecycle → the game → two choices
(flavour, tier) → the five flags → one slide per core flag → bonuses → house
rules → close.
